鸟哥的 Linux 与 ADSL 私房菜

学习之前
最近更新日期：2001/07/03
  • Linux 的来源：是由一个芬兰人 Linus Torvalds 于 1991 年 10 月开发出第一个核心之后，逐渐发展形成目前的情况。而由于他所发展的核心以 Unix 为基准去写的，所以也就通称叫 Linus's unix （Linux）。另外， Linux 是以 GPL（General Public License）的方式发表的，也就是 Linux 是开放性的，随时可以由网络上下载其源代码。只是当你修改 Linux 的原程序后，亦应该公开你的程序代码！
  • Linux 是最近几年来最受瞩目的操作系统之一，他会受到瞩目的原因主要是因为他是『free』的，就是免费的操作系统啦！然后他是开放性的系统，也就是你可以随时的取得程序的原始码，这对于程序开发工程师是很重要的！而且，虽然他是免费的，不过功能却很强大！另外， Linux 对于硬件的需求是很低的，这一点更造成他流行的主因，因为硬件的汰换率太快了，所以很多人手边都有一些很少在用的零件，这些零件组一组就可以用来跑 Linux 了，反正做一个工作站又不用使用到屏幕（只要主机就可以啰），因此 Linux 就越来越流行啰！（插个嘴，也就是因为 Linux 具有 1.硬件需求低、 2.架构开放、 3.系统稳定性及保密性工能够强、 4.完全免费，所以造成一些所谓『反微软联盟』的程序设计高手不断的开发新软件！以与 Microsoft 进行抗衡！）
  • 虽然 Linux 具有这样多的好处，但是他先天上有一个足以致命的地方，使他的普及率受到很大的限制，就是 Linux 需要使用 『指令列』 的终端机模式进行系统的管理！虽然近年来有很多的图形接口开发使用在 Linux 上面，但毕竟要熟悉 Linux 还是以指令列来使用是比较好的，因此要接受 Linux的玩家必须比较要能熟悉对计算机下指令的行为，而不是用鼠标点一点 icon 就行了！不过如果只是要架一些简单的小站呢？是不是大家都可以做的到？没错！其实只要对 Linux 做一些小小的设定就可以架站了，这也是我写这个网页的目的！如果你从来都没有摸过 Linux 这个咚咚，那也没关系，只要照着以下的步骤一步一步走下去，应该也可以架设完成属于你自己的网站的。
  • 我做这个小站的目的在于让大家可以轻易的就架设一个简易的 LinuxServer 来使用，所以文中并没有很有效的防火墙篇幅，因此，这个站里面的所有文件均不太适用于小规模以上的企业部门，仅适合一般家庭架设的网站，因为这里的技术文件保密性都不高....。不过，若您只是要架设一个 Mail server、FTP Server、Web Server 、NAT Server 等等的小站在自己家里的话，这里的文件应该还是挺合适的喔！
  • 话虽如此，但是一些基本的概念还是必须要有的，例如最基本的『计算机硬件配备』、『IP的分配与TCP/IP』的一些概念最好是有的，而且，具备简易的 DOS 概念也是必须的....
聪明学习
  • 我摸 Linux 只不过才一年，虽然架了几个小站在自己实验室及自己家里，但是对于 Linux 实在还是不懂，所以看不懂我这个小站里的数据的话，应该就是我不对啦！果真如此的话，欢迎替我指正，我会很乐意的接受的。因为我也是从完全不懂，在网络上 问人家时，一些高手还有点替我担心，有的则是认为我问的问题太白烂而不愿意回答我.....，不过大部分的同好还是很乐意的会回答您提出的问题的。
  • 所以，要学习 Linux 的话，一定要常上网站去看看有没有新的文件，然后在看一些技术文件的时候千万不要觉得麻烦，多看几遍给他看懂，这对你的学习一定会有所帮助的，尤其是基础的IP协定与一些基础的执行程序！
